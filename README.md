# Strapi & Nuxt MKT Page Demo 



## Setup

Make sure to install the dependencies within the frontend and backend folders:

```bash
# yarn
yarn install
```

## Development Servers

#### Frontend

```bash
yarn dev -o
```
#### Backend

```bash
yarn develop
```
